# Plan — Template Premium Secteur 23 : Industrie & Usine de Production

## Contexte
Génération d'un template web premium pour le secteur industriel (manufacturing, usine de production) basé sur les référence fournies. Le template doit être réaliste, crédible et personnalisable — destiné à être vendu comme produit démo.

## Livrable
Site web React complet (Next.js + Tailwind CSS) avec 9 pages publiques, composants réutilisables, design system industriel, et déploiement statique.

## Stages

### Stage 1 — Design System & Architecture
- Définir la palette de couleurs, typographie, spacing
- Créer les composants réutilisables (Hero, Cartes, Stats, Formulaires, Footer)
- Architecture des 9 pages du template
- **Skill**: vibecoding-webapp-swarm (Design System)

### Stage 2 — Génération des Assets Visuels
- Générer les images nécessaires : usine, machines, équipe en EPI, produits finis, portraits d'équipe
- **Outils**: generate_image (parallèle)

### Stage 3 — Développement des Pages
- Page d'accueil (Hero industriel + capacités + chiffres + certifications + CTA)
- Page Capacités (Production, Usinage, Assemblage, CQ, R&D)
- Page Secteurs Servis
- Page Qualité & Certifications
- Page Équipements
- Page Développement Durable (RSE)
- Page Carrières
- Page À Propos
- Page Contact avec formulaire de devis
- **Skill**: vibecoding-webapp-swarm (Development)

### Stage 4 — Polish & Déploiement
- SEO metadata, Schema.org markup
- Animations et transitions
- Responsive breakpoints
- Déploiement statique
- **Skill**: deploy_website

## Technologies
- Next.js (App Router) + TypeScript
- Tailwind CSS
- shadcn/ui components
- Framer Motion (animations)
- Lucide React (icônes)
